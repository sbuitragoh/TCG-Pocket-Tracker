# TCG Pocket Tracker

Simple project to track the collection inside TCG Pocket.
Works with:
- A3. Celestial Guardians
- A2b. Shining Revelry
- A2a. Triumphant Light
- A2. Space-Time Smackdown
- A1a. Mythical Island
- A1. Genetic Apex
